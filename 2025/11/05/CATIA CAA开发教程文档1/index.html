CATIA CAA开发教程文档(1)

发表于2025-11-05|更新于2025-11-06|CATIA二次开发CATIA教程
|浏览量:

1. 新建项目

1.1 创建工作空间（Workspace）

  • 点击 File->New CAA V5 Workspace

  • 输入工作空间名称（最好以Wsp结尾）

  • 选择Catia版本（V5R18_B18)

  • 点击Next

1.2 创建框架（Framework）

  • 选择Create new generic framework

  • 点击Finish

  • 输入框架名称（以Frm结尾）

  • 选择Implementation

  • 选择Development

  • 点击OK

  • 点击OK

1.3 加载编译环境

  • 点击CAA V5 Workspace

  • 选择Locate Prerequisite Workspace

  • 点击Add

  • 选择CATIA V5R18安装路径

  • 点击OK

  • 点击OK

  • 点击Close

1.4 添加模块（Module）

  • 输入模块名称（M结尾）

  • 选择Shared object

  • 点击OK

  • 点击OK

    注：模块文件夹会自动添加.m后缀

1.5 添加命令（Command）

  • 点击File->Add CAAV5 Item->CATIResource->Command

  • 输入Cmd名称（Cmd结尾）

  • 点击OK

  • 点击OK

1.6 添加对话框（Dialog）

  • 点击File->Add CAAV5 Item->CATIResource->Dialog

  • 输入Dialog名称（Dlg结尾）

  • 点击OK

  • 点击OK

1.7 创建外接程序（Addin）

  • 创建Addin模块（****AddinM）

  • 选择Addin模块为当前启动项

  • 点击File->Add CAAV5 Item->Component

  • 输入Component name（Addin结尾）

  • 点击Add

  • 点击…按钮

  • 选择CATIA安装路径（B18）

  • 移除Search…勾选

  • 选择实现接口模块，例如CATIPrtWksAddin（零件设计模块/曲面设计模块）

  • 其他常用模块

    • CATIDRWFRAMEAddin（工程制图设计模块）

    • CATIPRDWorkshopAddin（装配设计模块）

    • CATIAfrGeneralWksAddin（全局模块）

  • 点击OK

  • 点击OK

  • 点击OK

    注：如果需要修改或添加程序模块，可以通过修改addin对应的多个文件实现

    • xxxAddin.cpp    扩展组件对应的cpp文件
    • xxxFrm/CNext/code/dictionary/xxxFrm.dico    框架对应的dico文件

  • 添加Addin的.cpp中的Header声明宏

  • 在Addin的.h文件中public下添加函数声明

  • 添加对应的Addin模块引用到当前模块的Imakefile.mk，如果已经存在则不需要重复添加

  • 添加对应的模块组件到当前框架的IdentityCard.h，路径位于xxxWsp\xxxFrm\IdentityCard\IdentityCard.h，如果已经存在就不需要重复添加

  • 在Addin的.cpp文件中添加函数实现

1.8 添加资源文件

1.8.1 放置图标

  • 打开资源文件路径，一般为xxx\xxxWsp\xxxFrm\CNext\resource

  • 在resources文件夹内新建文件夹，名称为graphic

  • 在graphic文件夹内新建文件夹，名称为icons

  • 在icons文件夹内新建文件夹，名称为normal

  • 在normal文件夹中放置命令图标

1.8.2 放置文字资源

  • 打开资源文件路径，一般为一般为xxx\xxxWsp\xxxFrm\CNext\resource\msgcatalog

  • 在该路径下创建三个文件，分别命名为：

    • xxxAddin（与Addin类名一致，后缀为CATNls）

    • xxxAddinHeader（与宏声明的Header名称一致，后缀为CATRsc，内容是图标相关信息）

    • xxxAddinHeader（与宏声明的Header名称一致，后缀为CATNls，内容是文本显示信息）

  • 在该路径下创建Simplified_Chinese文件夹（中文环境资源文件夹）

1.8.3 更新资源文件

1.8.4 编译mkmk

1.9 添加环境变量-黑窗口输出

  • 点击工具->Runtime Environment Variables…

  • 环境变量名称：CNEXTOUTPUT

  • 环境变量值：CONSOLE

  • 点击Set

  • 点击OK

1.10 调试程序

1.10.1 使用Runtime Window

  • 点击窗口->Open Runtime Window

  • 输入CNEXT，回车启动

  • 点击工具->附加到进程

  • 选择CNEXT.exe附加

1.10.2 配置启动项

  • 左侧资源管理器选择待调试的模块右键设为启动项

  • 右键启动项目录选择属性->调试->命令

  • 选择CATIA路径下的CNEXT命令，路径一般在…./B18/win_b64/code/bin目录下

  • 如果以下提示，选择是
